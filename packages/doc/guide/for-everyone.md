## 🤓 What Exactly IS MagicSync?

### In One Sentence
MagicSync is a **social media management platform** that helps businesses and creators schedule posts, analyze results, and grow their audience — all from one dashboard!

### In More Words
Imagine you have a business with profiles on Facebook, Instagram, Twitter, LinkedIn, and TikTok. That's FIVE different apps you need to log into every day!

MagicSync puts ALL of those in ONE place. Write once, post everywhere!

## 🔧 Technical Overview

### Built With Modern Tech

| Component | Technology |
|-----------|------------|
| Framework | Nuxt 4 |
| UI | Nuxt UI v4 (Tailwind) |
| Auth | Better Auth |
| Database | Drizzle ORM + Turso |
| AI | Google Gemini 2.0 |
| Documentation | VitePress |

### Architecture

```
packages/
├── ui/           # Design system & components
├── auth/         # Authentication
├── db/           # Database layer
├── ai-tools/     # AI features
├── assets/       # Static assets
├── scheduler/    # Post scheduling
├── bulk-scheduler/ # CSV imports
├── connect/      # Platform OAuth
├── content/      # Blog/content
├── email/        # Email templates
├── templates/    # Post templates
├── site/         # Main application
└── doc/          # Documentation
```
