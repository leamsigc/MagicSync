# Introduction

MagicSync is the ultimate social media management platform designed for small businesses and power users. It allows you to manage multiple social media accounts from a single dashboard, schedule posts, analyze performance, and engage with your audience.

## Key Features

- **Multi-Platform Support**: Connect Facebook, Twitter, Instagram, LinkedIn, and more.
- **Smart Scheduling**: Auto-schedule posts for optimal engagement times.
- **Analytics**: Deep dive into your performance metrics.
- **Team Collaboration**: Work together with your team members.
- **Extensible**: Built on a modular architecture with Nuxt layers.

## Architecture

MagicSync is built as a monorepo using Nuxt 3 and pnpm workspaces. It consists of several packages:

- `packages/ui`: The core UI component library.
- `packages/auth`: Authentication and user management.
- `packages/scheduler`: The core scheduling logic.
- `packages/doc`: This documentation site.
