# Installation

## Prerequisites

- Node.js 18+
- pnpm 8+

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/leamsigc/magicsync.git
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

## Docker Support

You can also run MagicSync using Docker:

```bash
docker-compose up
```
