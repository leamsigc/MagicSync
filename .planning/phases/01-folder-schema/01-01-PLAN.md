Add folder table to the Drizzle schema with the following structure:

```typescript
// Folders table - adjacency list pattern for unlimited nesting
export const folders = sqliteTable('folders', {
  id: text('id').primaryKey(),
  name: text('name').notNull(),
  parentId: text('parent_id').references(() => folders.id, { onDelete: 'cascade' }),
  ownerId: text('owner_id').notNull(), // user ID or 'global'
  isGlobal: integer('is_global', { mode: 'boolean' }).notNull().default(false),
  createdAt: integer('created_at', { mode: 'timestamp' }).notNull().default(new Date()),
  updatedAt: integer('updated_at', { mode: 'timestamp' }).notNull().default(new Date())
})

export type Folder = InferSelect<typeof folders>
export type FolderInsert = InferInsert<typeof folders>
```

Key points:
- parentId creates adjacency list (self-referential FK with cascade delete)
- isGlobal distinguishes global (visible to all) from per-user (private)
- ownerId stores user ID or 'global' for global folders
- Default functions from schema.ts for id generation
